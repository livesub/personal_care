🏗️ 장애인 활동지원 플랫폼 기능 정의서 (Final Summary)
1. 사용자(장애인 및 보호자) 파트
최우선 과제: 서비스의 실시간 모니터링과 바우처 잔액의 투명한 관리.
•	대시보드: 이번 달 총 바우처 시간 중 사용량과 잔여량을 그래프로 시각화하여 확인.
•	실시간 활동 확인: 지원사의 출근 여부 및 현재 수행 중인 활동 상태를 실시간으로 모니터링.
•	복약 확인 알림: 지원사가 약을 복용시킨 후 남긴 기록(사진 등)을 통해 제시간에 복약했는지 실시간 확인.
•	활동 일지 승인: 지원사가 작성한 일지를 확인하고 디지털 서명 또는 확인 버튼을 클릭하여 최종 확정.
•	긴급 호출 및 SOS: 사고 발생 시 보호자와 센터 관리자에게 즉시 위치 알림 전송.
•	만족도 피드백: 서비스 종료 후 별점이나 의견을 센터에 전달하여 서비스 품질 관리.
2. 활동지원사 파트
최우선 과제: 정확한 근태 기록 및 복잡한 행정 업무의 간소화.
•	GPS 출퇴근 인증: 이용자 자택 반경 내에서만 서비스 시작/종료 버튼이 활성화되어 정확한 근태 기록.
•	디지털 케어 가이드: 서비스 시작 전 이용자의 특이사항(마비 방향, 알레르기 등)을 팝업으로 확인하여 실수 방지.
•	음성 인식(STT) 업무 일지: 손이 자유롭지 못한 현장 특성을 고려해 음성으로 일지를 작성하고 자동으로 텍스트 변환.
•	이동지원 경로 기록: 병원 이동이나 산책 시 이동 시작/종료를 눌러 실제 이동 경로와 거리를 지도에 기록.
•	타임스탬프 카메라: 특이 사항이나 사고 발생 시 조작 불가능한 시간/위치 정보가 포함된 사진으로 증빙 기록.
•	복약 지도 체크리스트: 정해진 시간에 푸시 알림을 받고, 복약 후 사진 촬영이나 체크를 해야 업무 완료 가능.
•	급여 계산기: 실제 근무 시간을 바탕으로 주휴/가산수당이 포함된 예상 급여 자동 계산.
•	오늘의 팁 & 리워드: 장애 유형별 맞춤 팁을 확인하고, 성실 근무 시 적립되는 감사 포인트로 소속감 고취.
3. 관리자(센터 운영자) 파트
최우선 과제: 부정수급 방지 및 행정 데이터의 자동화.
•	실시간 모니터링 대시보드: 현장에 나간 지원사의 수와 실시간 위치를 확인하여 이상 징후 감지.
•	부정수급 방지 시스템: 자택 이외 장소에서의 태깅이나 시간 중복 시 즉시 경고 알람 수신.
•	스케줄 및 대차 관리: 이용자와 지원사를 연결하고, 지원사 부재 시 소속 내 가용 인력을 즉시 찾아 매칭.
•	정부 보고용 엑셀 자동화: 앱 데이터를 공공기관(행복e음 등) 보고 양식에 맞춘 엑셀 파일로 원클릭 생성.
•	스케줄 변경 승인: 지원사나 이용자의 시간 변경 요청을 앱에서 확인하고 승인하여 정산 데이터에 반영.
•	메시징 시스템: 전체 지원사를 대상으로 긴급 공지나 교육 안내를 푸시 알림으로 발송.


1. 사용자 관리 (가장 우선순위)
관리자가 앱 사용자를 생성하고 권한을 제어하는 기능입니다. 
•	신규 사용자 등록: 장애인(이용자) 및 활동지원사의 아이디와 초기 비밀번호(4자리 이상)를 생성합니다. 
o	DB 규칙: is_first_login 값을 1로 고정하여 등록하고, created_at에 등록 시간을 필수 저장합니다. 
•	아이디 중복 체크: 관리자가 입력한 login_id가 DB에 이미 존재하는지 실시간으로 확인합니다. 
•	비밀번호 초기화: 사용자가 비밀번호를 분실했을 때 관리자가 임시 비밀번호로 강제 재설정해주는 기능입니다.
•	소속 센터 매칭: 각 사용자를 특정 센터나 지찰에 귀속시켜 관리 범위를 설정합니다. 
2. 매칭 및 스케줄 관리
이용자와 지원사를 연결하고 업무 시간을 배정합니다. 
•	활동 매칭: 특정 장애인 이용자에게 담당 활동지원사를 배정합니다. 
•	서비스 제공계획서(디지털 가이드) 작성: 이용자의 장애 유형, 주의사항, 케어 방법 등을 입력하여 앱의 '케어 가이드'로 전송합니다. 
•	스케줄 확정 및 변경 승인: 현장에서 요청온 시간 변경 건을 확인하고 최종 승인하여 바우처 정산에 반영합니다. 
3. 모니터링 및 부정수급 관리
현장의 상황을 실시간으로 파악하고 이상 징후를 감지합니다. 
•	실시간 근무 현황: 현재 출근 중인 지원사 수와 GPS 기반의 대략적인 위치를 지도로 확인합니다. 
•	부정수급 알람 관리: 지정된 장소(이용자 자택 등)가 아닌 곳에서 태깅을 시도한 이력을 확인하고 조치합니다. 
•	포토 업무 일지 및 사고 보고: 지원사가 현장에서 찍어 올린 타임스탬프 사진과 특이 사항을 실시간으로 확인합니다. 
4. 행정 및 데이터 관리 (현업 핵심)
복잡한 서류 작업을 자동화하는 기능입니다. 
•	바우처 잔액 관리: 이용자별 바우처 잔여 시간을 모니터링하고 정산 오류를 체크합니다. 
•	정부 보고용 엑셀 자동 추출: 행복e음 등 공공 시스템 양식에 맞춰 활동 기록 데이터를 엑셀로 원클릭 생성합니다. 
•	이동 경로 증빙 출력: 지원사가 기록한 이동지원 경로(GPS) 데이터를 교통비 청구용 증빙 자료로 출력합니다
