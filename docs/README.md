# 📑 [Project Master Guide] 장애인 활동 지원 플랫폼: PERSONAL CARE

## 🛠️ 개발 환경 (Environment)
* **라라곤:** v8.5.0 (260125)
* **PHP:** php-8.5.1-Win32-vs17-x64
* **Framework:** Laravel 12 (Eloquent ORM 필수)
* **Database:** mysql-8.4.8-winx64
* **Web Server:** nginx-1.29.5
* **Flutter:** flutter_windows_3.38.9-stable
* **IDE:** 안드로이드 스튜디오 / 커서 AI
* **Tool:** Git 설치 / Flutter SDK / Composer
* **AI:** 커서 AI(바이브 코딩)

## 1. 시스템 기초 및 개발 환경 (Technology Stack)
프로젝트의 뼈대가 되는 기술과 버전 정보입니다. 초보자분들을 위해 각 기술의 역할도 상세히 풀었습니다.

* **개발 플랫폼:** 단일 코드베이스로 **Web(크롬 등), Android(갤럭시), iOS(아이폰)**를 모두 지원합니다. 어디서든 똑같은 화면을 볼 수 있습니다.
* **프론트엔드 (화면): Flutter (Dart 최신 버전)**
    * 구글이 만든 최신 기술로, 버튼 하나, 글자 하나까지 모든 기기에서 동일하게 보이도록 설계되었습니다.
    * shared_preferences 패키지를 사용하여 로그인 정보를 안전하게 기억합니다.
* **백엔드 (엔진): PHP 8.5 / Laravel 12 (라라벨)**
    * 가장 최신 버전의 엔진을 사용하여 속도가 매우 빠르고 보안이 강력합니다.
    * 복잡한 급여 계산을 담당하는 두뇌 역할을 합니다.
* **데이터베이스: MySQL 8.x**
    * 보호사와 관리자의 정보를 각각 다른 Table에 나누어 보관하여 정보가 섞이지 않게 관리합니다.
* **보안 아키텍처:**
    * **Sanctum:** 로그인 후 생성되는 '디지털 입장권'입니다. 새로고침을 해도 로그인이 유지되게 해주는 핵심 기술입니다.
    * **Argon2id & AES-256-CBC:** 비밀번호와 개인정보를 은행 수준으로 암호화하여, 관리자조차 비밀번호 원문을 볼 수 없게 보호합니다.

## 2. 사용자별 핵심 서비스 흐름 (Workflow)

### A. 보호사 (Helper): 계정 생성부터 급여 확인까지
1. **아이디 생성:** 관리자가 보호사의 휴대폰 번호로 초기 계정을 만들어줍니다.
2. **보안 강화 (10자 비밀번호):** 최초 로그인 시 **반드시 10자리 이상(영문+숫자+특수문자)**의 새 비밀번호로 바꿔야 합니다. 변경 전에는 다른 어떤 기능도 쓸 수 없게 차단됩니다.
3. **로그인 가드:** 보호사가 아닌 사람(관리자 등)이 보호사 페이지로 접속하려 하면 시스템이 즉시 감지하여 차단합니다.
4. **실시간 업무 수행:**
    * **업무 시작:** 정해진 시간 전후 20분에만 [시작] 버튼이 나타나 정직한 근무를 유도합니다.
    * **업무 종료:** 오늘 이용자님의 상태를 적는 '업무 일지'를 써야 종료가 가능합니다.
    * **조기 종료 사유 강제:** 예정보다 일찍 퇴근하려고 버튼을 누르면 **"왜 일찍 종료하시나요?"**라는 팝업이 뜨며, 사유를 입력해야만 기록이 저장됩니다.
5. **급여 카운트업:** 홈 화면에서 이번 달 내가 받을 예상 금액이 0원부터 목표 금액까지 촤르륵 올라가는 애니메이션이 나타나 업무 의욕을 높여줍니다.

### B. 관리자 (Admin): 임시 계정 폭파와 정식 운영
1. **임시 계정 파괴 (Hard Delete):** 초기용 아이디(admin_센터코드)로 접속해 정식 관리자를 등록하는 순간, 그 임시 아이디는 서버에서 영구히 삭제되어 보안 허점을 없앱니다.
2. **등급별 권한:** 모든 걸 관리하는 '슈퍼 관리자'와 정산만 못 보는 '일반 스태프'로 명확히 나뉩니다.

## 3. 관리자 메뉴별 상세 기능 (누락 없는 전수 조사)

### ① 대시보드 (실시간 통합 현황판)
* **통계 카드:** 전체 보호사/이용자 수, 오늘 일정 총합, 그리고 현재 실제로 이용자님을 케어 중인 인원을 실시간 숫자로 보여줍니다.
* **라이브 모니터링:** 오늘 일정을 시간순으로 나열하고 상태별 색상 뱃지(대기-회색, 진행중-초록, 완료-파랑)를 달아 센터 가동률을 한눈에 봅니다.

### ② 회원 관리 (데이터 격리와 다중 소속)
* **보호사 다중 소속:** 한 명의 보호사가 여러 센터에서 일해도 정보를 새로 등록할 필요 없이 **'소속 센터 추가'**만으로 모든 일정을 통합 관리합니다.
* **이용자 관리:** 주민번호를 넣으면 성별이 자동 판별됩니다. 이용자님의 바우처 남은 시간과 비상 연락처를 밀착 관리합니다.
* **계정 잠금 해제:** 비번을 5번 틀려 앱이 잠긴 보호사를 위해 관리자가 버튼 하나로 즉시 풀어줍니다.

### ③ 매칭 관리 (지능형 일정 설계)
* **전역 중복 체크 (Conflict Check):** 일정을 짤 때, 해당 보호사가 다른 센터 일정과 단 1분이라도 겹치는지 시스템이 전수 조사하여 경고를 보냅니다.
* **맞춤 시급 설정:** 특정 매칭(예: 공휴일 근무)에 대해서만 관리자가 수동으로 시급을 수정할 수 있습니다.
* **조기 종료 기록:** 보호사가 일찍 퇴근하며 적은 사유를 리스트에서 즉시 확인하고 행정 근거로 사용합니다.

### ④ 실시간 모니터링 (현장 관리)
* **지각 알림:** 시작 10분이 지났는데도 버튼을 안 누른 보호사는 리스트 맨 위로 빨갛게 강조되어 올라옵니다. 관리자는 바로 전화를 걸어 상황을 파악할 수 있습니다.
* **상태 필터링:** [전체, 서비스 중, 시작 전, 조기 종료, 완료] 탭을 눌러 원하는 상태만 골라봅니다.

### ⑤ 행정 및 정산 (1초 자동 계산)
* **분(Minute) 단위 합산:** 실제 시작과 종료 시간의 차이를 분 단위로 계산하여 총 근무 시간을 산출합니다. (예: 1시간 40분 → 1.67시간)
* **자동 수당 계산:** 밤 10시 이후 야간 수당이나 공휴일 1.5배 할증을 사람이 계산하지 않고 시스템이 자동으로 계산합니다.
* **정산 리스트:** 보호사 지급액과 나라에 청구할 돈을 엑셀 형태로 완벽하게 뽑아줍니다.

### ⑥ 운영 설정 및 실시간 알림
* **스태프 등록:** 나를 도와줄 직원을 추가합니다. 단, 보안을 위해 스태프는 '관리자'를 추가할 수 없고 '정산' 메뉴도 볼 수 없습니다.
* **실시간 공지 팝업:** 관리자가 웹에서 공지를 올리면, 보호사 앱 홈 화면에 해당 내용이 자동으로 튀어나오는(Auto-popup) 실시간 연동이 구현되었습니다.

## 4. 프로젝트의 독보적 완성도 (Summary)
* **철통 보안 라우팅:** 주소창에 강제로 주소를 치고 들어와도 권한이 없으면 무조건 쫓아내는 보안 가드가 작동합니다.
* **새로고침 무적:** 인터넷 창을 새로고침해도 로그인이 풀리지 않아 업무의 연속성을 보장합니다.
* **다국어 지원:** 현재 한국어, 영어, 베트남어로 작업되어 있습니다.
* **돋보기 지원:** 시력이 낮은 분들을 위한 글자 크기 조절 기능.
* **차후 SMS 기능과 SNS 회원 가입 고려중
